CNT4713 Final Project - Static Feed Application

Implemented by Shaan Khan, Panther ID 6112001

This static feed application serves files from an AWS S3 bucket; it was written in Python and makes use of the Flask framework. It was configured and deployed to the cloud via AWS ElasticBeanstalk as an EC2 instance.

Auth0 was used for authentication via an accessory application also written using Python and Flask. This application, which handles logins and logouts, was configured and deployed to the cloud via Heroku.
